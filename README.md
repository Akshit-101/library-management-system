# Library Management System

A simple library management system for managing books, tracking members, viewing transaction histories, and issuing books.

## Features

- **Book Management:** View and manage the collection of books.
- **Member Management:** View and track library members.
- **Transaction History:** View transaction logs and track book issues.
- **Issue Books:** Issue books to members and track their status.

## Tech Stack

- **Frontend:** React + TailwindCSS
- **Backend:** Express
- **Database:** Postgres
- **Containerization:** Docker

## Getting Started

### How to Run

1. **Clone the repository** and navigate to the project root directory:
   ```bash
   cd library-management-system
   ```

2. **Start the application** using Docker Compose:
   ```bash
   docker compose up --build -d
   ```

3. **Access the application** in your browser at:
   ```
   http://localhost
   ```
